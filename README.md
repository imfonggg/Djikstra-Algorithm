# Djikstra-Algorithm

Final Project: learn how Djikstra's Algorithm works and imeplement the algorithm as a team to find the shortest path possible between destinations around USF campus.

*Steps:*
- Pick starting node
- Initialize tables with values corresponding to the shortest path, set to infinity (very big value) if not visited
- Check neighbor nodes and update table with distance and nodes visited
- Repeat until end node is reached
- Return the shortest path
